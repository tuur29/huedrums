## Features

Current Features:
- Turn a lamp on as long as you keep your finger on the drum
- Edit colors and brightness mid-drumming
- Strobe your phone's screen
- Easily add a baseline
- Move and resize your drums
- Short tutorial
- Multitouch support
- ...
